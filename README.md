# User Analytics Dashboard

## Project Overview
This project is Phase 1 of a university DevOps end-term assignment. It creates a professional local Flask dashboard that reads a locally stored user dataset and presents interactive statistics.

## Folder Structure
- app.py: Flask application entry point
- analysis.py: Dataset loading and metric generation
- templates/index.html: Dashboard layout
- static/style.css: Dark blue dashboard styling
- static/script.js: Counter and Chart.js rendering logic
- data/u.user: Local dataset file
- screenshots/: Placeholder directory for screenshots

## Requirements
- Flask
- Pandas

## Installation
```bash
pip install -r requirements.txt
```

## How to Run
```bash
python app.py
```
Then open http://127.0.0.1:5000 in your browser.
